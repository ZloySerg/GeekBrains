## Добавление веток

Для создания новой ветки необходимо проверить сначала положение в главной ветке:

**git branch**

должна быть звездочка напротив *master*  
\* <span style="color:lightgreen"> master</span>


Затем создаете новую ветку:  
**git branch _Name_**
